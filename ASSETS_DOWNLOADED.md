## Notes
- Most assets are 16x16 or 32x32 pixel format suitable for top-down view
- Some larger sprite sheets contain multiple characters/items in a grid
- office_furniture.png (1024x1024) is a comprehensive tileset
- characters.png contains multiple character sprites suitable for top-down games
- LPC (Liberated Pixel Cup) assets follow consistent pixel art conventions similar to Stardew Valley style
